# 💰 Finance Analysis Dashboard | Power BI

## 📖 Project Overview

The Finance Analysis Dashboard is an interactive Power BI project that provides comprehensive insights into
financial transactions, customer behavior, and business performance. The dashboard enables users to monitor
key financial metrics, analyze transaction trends, and make data-driven decisions through dynamic visualizations and filters.


## 🎯 Key Objectives

- Track overall financial performance.
- Monitor transaction volume and value.
- Analyze customer segments and occupations.
- Evaluate transaction success, failure, and pending status.
- Identify top-performing states and transaction types.
- Support business decision-making with interactive analytics.


## 📊 Dashboard Features

### KPI Cards
- Total Amount
- Total Transactions
- Average Transaction Value
- Total Fees
- Total Tax

### Visualizations
- Monthly Total Amount Trend
- Transaction Status Distribution
- Customer Segment Analysis
- State-wise Revenue Analysis
- Occupation-wise Analysis
- Transaction Type Analysis Matrix

### Interactive Filters
- Year Filter
- Dynamic Metric Selector
- Occupation Filter
- Merchant Category Filter
